bonnard-cli provides a straightforward way to access analytics tools. It combines a server, dashboards, and a command-line interface (CLI) to help you work with data. You do not need programming skills to use it.

## 🖥️ What is bonnard-cli?

bonnard-cli is a tool designed to connect to modern analytics systems. It helps you view data dashboards, manage metrics, and use a semantic layer to organize your information. The tool works with platforms like BigQuery, Snowflake, and Databricks. It is made for business users who want clear insights without technical setup.

This tool includes:

- A server that runs on your computer or network (MCP server).
- Dashboards for easy data visualization.
- A command-line interface to manage data settings and reports.
- Support for data platforms such as BigQuery, Snowflake, and more.

## ⚙️ System Requirements

Before you start, check that your computer meets these needs:

- Operating System: Windows 10 or later
- Processor: Intel i3 or equivalent, 1.5 GHz or faster
- RAM: At least 4 GB
- Disk Space: Minimum 200 MB free
- Internet Connection: Required for initial setup and data updates
- Optional: Access to data sources like BigQuery or Snowflake (not mandatory for basic use)

## 💻 Using the Command-Line Interface (CLI)

The CLI lets you perform advanced actions, such as:

- Connecting to new data sources
- Running queries
- Managing semantic data layers

To use the CLI:

1. Open Command Prompt (search for "cmd" in the Start menu).
2. Navigate to the folder where bonnard-cli is installed. For example:

   ```
   cd C:\Users\YourName\bonnard-cli
   ```

3. Run commands with this format:

   ```
   bonnard-cli [command] [options]
   ```

For example, to see all available commands:

```
bonnard-cli help
```
